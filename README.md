# EKPA Courses

## Setup

1. Install Docker Desktop
2. On your terminal, run the following command:
   ```
   docker compose up --build --detach
   ```

## Configuration

URLS:

- metabase: http://localhost:9991
- pgAdmin4: http://localhost:9993

Postgres

- username: postgres
- password: password
- database: university
- host: localhost
- port: 5432
